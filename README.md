# PromoLog

PromoLog is a React-based work logging app that tracks tasks and calls with live durations.

## Features

- Start and stop task timers
- Start and stop call timers
- Live duration updates while active
- Edit task descriptions in-app
- Session persistence using sessionStorage
- Export logs to CSV

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
